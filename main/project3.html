Ruby on Rails System for matching mentors with mentees.

The Mentor-Mentee Matching System is a web application I developed to facilitate the connection between mentees and mentors in university settings. Leveraging technologies such as Ruby, Sinatra, and SQLite 3, the system offers a secure platform for students to find mentors in their respective subjects. With its intuitive interface and efficient matching algorithms, the system streamlines the process of mentorship, empowering students to easily find the right tutors and enhance their academic journey. Check out the GitHub repository for more details and code samples.
